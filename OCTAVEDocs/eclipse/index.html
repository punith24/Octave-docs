    Eclipse Installation

    • Download Eclipse from its official download page.Download eclipse as per requirement. All the recent releases are available on the page.

    Note

    Make sure you opt and download the appropriate eclipse IDE as per your system configuration. There are two download links available for 64 bit windows operating system and 32-bit windows operating system.

    • It might take a few minutes to download the complete zip folder.

    • Once downloaded, copy the folder and place it in the desired location on your file system.

    • Extract the zipped folder, a folder named as eclipse can be seen. The folder embodies all the required application and source files.

    • Launch the eclipse IDE using “eclipse.exe” residing inside the eclipse folder. Refer the above illustration for the same.

    • The application will prompt you to specify the workspace location. Workspace is that location where all your eclipse projects will be residing. Enter/Browse the desired location or the user can simply opt for the default location and click on the OK button.
